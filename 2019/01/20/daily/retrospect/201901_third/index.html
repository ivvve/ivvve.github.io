회고

  • 토이 프로젝트

포트폴리오 겸 새로운 재미를 찾을 겸 토이 프로젝트를 진행하고 있다.
그냥 배우기만 했으면 몰랐을 것들도 새로 알 수 있었고
무엇보다 내가 만들고 싶은 것을 만든다는 것이 진짜 개발의 재미가 아닐까 생각한다.
재미도 있고 실력도 늘고 회사에서는 사용하지 않아서 못쓰는 기술들도 사용해 볼 수도 있고 하길 잘 했다는 생각이 든다.

  • Terracotta server

회사에서 Ehcache clustering을 알아보라고 지시해서 Terracotta server에 대해 알아봤다.
오픈소스 버전, 상업 버전이 있는데 회사에서는 오픈소스 버전을 찾아보라고 했지만
이 차이에 대해 정확히 선 긋고 설명한 자료가 공식 홈페이지에도 없어서 삽질을 많이 했다.
Ehcache 자료는 많지만 Terracotta server에 대한 한국어 거의 없다시피 해서 공식 홈페이지, document를 읽고 또 읽었다.
평소 document는 안읽혀서 잘 보질 않았는데 덕분에 document를 전보다는 잘 볼 수 있을 것 같다.
삽질을 했으니까 이에 대한 것들도 정리할 겸 포스팅해야겠다.

  • SQL 첫걸음

SQL 쓰긴 썼고 나름 복잡한 쿼리도 만들어 보긴 했는데 그래도 전부터 SQL 기초가 모자른다고 생각했었다.
그래서 전 부터 평이 좋았던 SQL 첫걸음을 저번주 부터 출퇴근 시간에 지하철에서 보기 시작했는데 쉽게 읽혀서 금새 다 읽었다.
물론 기초 책이긴 하지만 나 처럼 어느정도 SQL을 써봤지만 기초가 모자르다고 생각하면 추천해줄 만한 책이다.
다음 주는 다시 보면서 리뷰하는 시간을 가져야겠다.
