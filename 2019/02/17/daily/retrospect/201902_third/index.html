  • GDG 델리만주 세미나

GDG에서 학생들을 위한 세미나를 개최했는데 나도 최근 이직을 준비하면서 들을만한게 있나해서 참여하게 됐다.
아무래도 현재 학생이 아니다 보니 그렇게 와닿는 내용은 없었지만
취업 시장에서 파이썬의 수요 및 github에 정적페이지 올리기 두 가지는 얻은 것 같다.
또한 나 보다 어리지만 나 보다 잘나가는 개발자의 이야기를 들으면서 경쟁심이 나서 이번 주말 더 열심히 공부했던 것 같다.
(덤으로 끝나고 집가는 길에 Nomad Coder의 Nicolas를 만났다. 한국은 정말 좁은 것 같다…)

  • Python

전부터 다른 언어를 공부하는 것에 대해 많이 생각해왔다.
단지 취업만이 아니라 다른 언어를 공부하면서 다른 것들을 얻고 싶었다.
프로그래밍적인 생각을 넓힌다던지 해당 언어에 특화된 기능을 경험해본다던지를…
파이썬은 사용되는 곳도 많고 웹 개발 시장에서도 많은 수요가 있기 때문에 코틀린, go와 함께 올해 배울 언어 목록에 포함시켜놨는데 GDG 세미나를 듣고 파이썬의 시장 수요가 생각외로 많기 때문에 이직 + 공부를 위해 파이썬으로 확정했다.
사실 제작년 첫 프로그래밍을 파이썬으로 시작했었는데 덕분에 수월하게 배우고 있다.
블로그에 정리하면서 파이썬도 지금 자바를 쓰는 것 만큼 능숙하게 사용하도록 해야겠다.
