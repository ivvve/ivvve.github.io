Postman - 8. Pre-request Script

2019-05-19
postman

요청 때마다 랜덤한 정수를 사용하고 싶다면 어떻게 할까?
Postman은 Pre-request Script라는 기능이 있어 요청 전에 특정 스크립트 코드를 실행시킬 수 있다.

Pre-request Script 탭을 클릭해서 앞서 랜덤한 정수값을 생성시킨 뒤 전역 변수 id에 담는다.

이제 전역 변수 id를 사용해서 요청을 보내면 된다.

Pre-request Script 기능을 사용하기 위해 스크립트로 랜덤한 정수를 생성했지만 사실 이런 랜덤한 정수를 생성하는 built-in 기능이 있다.
