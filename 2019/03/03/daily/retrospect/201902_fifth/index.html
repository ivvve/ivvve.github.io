TDD 실천법과 도구를 공부했는데 제대로 기록을 안해놔서 어디까지 공부했는지를 잘 모르겠다.
기록을 잘 하자…

  • method간 synchronized 관련 고찰…

그 동안 개발을 하면서 줏어 들은 기술 동냥으로 나름의 로직을 짜는데 많은 노력을 기울였다.
(결국 ReentrantReadWriteLock 하나로 끝나긴 했지만…)
역시 개발은 많은 삽질과 경험, 그리고 다른 개발자들과의 교류가 중요하구나를 깨달았다.

  • 3.1절 100 주년 기념 10km 마라톤

개발 관련은 아닌데 내 인생에 있어 하나의 의미있는 이벤트가 아니였나싶다.
10km를 달리면서 내가 잘 하고 있나와 앞으로의 방향성을 곰곰히 생각해보았고 또 포기하지 않고 달리면서 인내력을 좀 길렀던 것 같다.
그리고 3.1, 3.2일 이틀 앓아누우느며 나의 허약함에 반성을 하였고 운동의 필요성을 느꼈다….
이래저래 의미있는 이벤트였다.

이번 연휴동안에 개발에 많은 집중은 하지 못했지만 나름 여러가지를 새로 경험해 볼 수 있어 뜻 깊은 3일이었지 싶다.
refresh 했다는 마음으로 앞으로 있을 clean code 강의를 위해서 다시 달려야겠다.
