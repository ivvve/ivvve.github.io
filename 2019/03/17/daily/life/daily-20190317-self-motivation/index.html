Self motivation

2019-03-17

I think the most important thing for developers is self motivation.
That’s because we have to keep our passion on programming and coding.
And I don’t think it’s easy because there are so many technologies to learn and new techs keep coming out. This can make us overwhelmed.
So keeping getting self motivation is one of the key factor of professonals.

As for me, I usually get self motivation from other capable developers.
I mean compared to them, I’m so novice.
So I feel like this, ‘I wanna be like him. And someday, I’ll surpass him!’

And also amazing tech is also makes me excited.
When I first learn about React, it’s very amazing.
Because I used only jQuery and you know manipulating DOM with only jQuery is kinda painful.
But React, it’s very simple to handle DOM using components.
Even though I’m a back-end developer, I was able to get interest on it.

These dyas, I’m learning TDD and clean code.
And it also give me self motivation to make better codes!

I hope every developers can find their own fun from programming like me.
Then our IT industry in Korea can be healthier and better.

This is a picture that I took on my way to a cafe.
Just I liked this view somehow. :)
