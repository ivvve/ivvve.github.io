회고

  • TDD
    일하지 않는 모든 시간을 볼링에 투자했는데 더 이상 진행이 되질않았다.
    나름 힌트도 보고 이것저것 다 생각해보고 해봤는데 잘 되질 않아서
    아직 내공이 부족한것 같아 일단은 잠시 내려놓고 처음부터 다시 복습을 하고 있다.
    그 동안 배웠던 경험 덕분에 이전보다 더 수월하고 코드도 좀 더 간결하게 구현을 할 수 있었다.
    꼭 볼링을 끝낼 수 있도록 계속 노력해야겠다.

  • Cloud Native Java
    책을 사 놓고 구석에 쳐박아 놓았던 책인데 일단 TDD 강의가 끝나 읽기 시작했는데
    앞장에 클라우드에 대한 최근의 이야기와 클라우드에 대해 잘 설명해 놓았고
    Pivotal의 서비스를 사용하는 방법도 잘 나와있었다.
    두꺼운 책이지만 꾸준하게 해서 꼭 끝을 봤으면 좋겠다.

  • Think Data Structures
    이것도 구석에 쳐박아 놓았던 책인데 계속 읽어야지 하면서 미뤘는데 다시 읽고 있다.

현재 책을 3권 읽는 중인데 책을 분산으로 읽는 이유는 한 권이 지루할 때 refresh를 시켜줄 수 있을 것 같고, Cloud Native Java와 같이 기술 위주의 서적과 Think Data Structures와 같은 기본 서적을 번갈아 읽어가면서 동시에 서로 성격이 다른 공부할 수 있을 것 같아서이다.
나름 공부법에 대해서 테스트 중이다.

잘한 점

  • 영어 학원 스터디 도중에 스터디리더가 불가피하게 스터디를 참여할 수 없게 되었는데, 수업 보조로서 스터디 인원간 의견을 잘 조율해서 새로운 진행방식을 정했다.
  • TDD 미션을 다시 복습하였다. (다음 목표는 개인 프로젝트를 TDD로 진행하는 것)

반성할 점

  • 운동을 이번 주에 피곤하다는 핑계로 빼먹은 날이 많았다.
  • 주말에 많은 시간을 너무 낭비하는 것 같다. 스터디를 참여하던지 무엇을 하던지 일단 외부로 나갈 수 있게 해야겠다.

새로운 주는

  • 기존 서비스에서 계속 추가사항이 들어오고 있어서 개인적으로 공부할 시간이 많을 지 잘 모르겠다. 그래도 지금 공부하는 것들 새로 공부를 시작하는 것들을 계속 해가면서 계속 성장했으면 한다.
  • 다음 주말은 좀 더 알차게 시간을 보내도록 해야겠다.
