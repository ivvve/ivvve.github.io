회고

  • Redis

기존 서비스의 확장에 대비해 레디스 공부를 다시 시작했다.
당시 내공이 거의 전무해서 책을 읽으면서도 이해를 못했는데 이제 읽으면서 어느정도 이해를 할 수 있다는게 나름 많이 성장한 것 같다.
이게 배우는 즐거움이 아닐까 싶다. 앞으로도 업무를 하면서도 계속 배우는 재미를 느끼도록 하자.

  • 학원 스터디 관리 프로젝트

영어학원에서 반쯤 스터디 리더의 역할을 하고있다.
스터디 리더로서 팀원들 관리, 숙제 관리 등을 해야하는데 기존 몇 년 동안 계속 수작업으로 모든 관리가 이뤄졌다.
개발자다보니 이런 수작업이 번거롭게 느껴졌고 재미반 필요반으로 새롭게 프로젝트를 시작했다.
요즘 바쁘다보니 이건 아마 몇 달 걸릴 것 같지만 누군가에게 도움이 되는 소프트웨어를 개발하는게 개발자의 존재 의미 중 하나가 아닐까? :)

저번주의 목표에서

  • 회사에서 새로운 추가사항이 들어오고 꽤 큰 업데이트라서 평일에 시간을 많이 내질 못했다. 하지만 새로운 목표를 새우고 퇴근 후에도 공부하려고 노력했다.
  • 주말엔 빈둥대느라 카페에 일찍 가진 못했지만 주말에 카페에서 공부도 하고 여러 생각도 할 수 있었다.

잘한 점

  • 운동할 겸 서울대입구역에서 신림까지 걸어오는데 이북 TTS 기능을 사용하면서 책을 들으려고 노력했다.
  • 일할때 새로운 마음가짐을 갖고 최선을 다하려고 노력했다. 프로답게 업무를 대하도록 더 노력하자.

반성할 점

  • CS 강의를 제대로 시작하기로 해놓고 제대로 유지하질 못했다.

새로운 주는

  • 새로운 업무가 까다롭고 시간이 부족해도 프로답게 더 좋은 서비스 개발을 할 수 있게 집중하자.
  • CS 강의 무조건 계속 듣자!
  • 주말에 너무 늘어져서 잠을 필요 이상으로 잤다. 다음 주에는 평일만큼은 아니더라도 수면시간을 적당히 가져야겠다.
